python学习笔记（1）---模块的导入与重载

  • python
python

每一个以扩展名’py’结尾的Python源代码都是一个模块，其他文件可以通过导入一个模块而读取该模块的内容。

基本实例：script1.py:

可以通过’import’来运行上诉的script1.py:

但只有第一次导入时候才会运行script1.py，以后的导入都不会再次的运行，即使在另一个窗口中改变并保存了script1.py的源代码。

如果需要在一次python会话中再次运行已导入的文件（如文件的源代码已改变），需要用到reload函数。它在python2.x是内置函数，在python3.x被移到了imp标准库中。

‘reload’函数载入并运行了模板的最新版本代码。

需要注意，reload函数需要获得的参数是一个已经加载了的模块对象的名称，所以在重载之前需确保已经导入了这个模块。reload是个函数，需要在模块名前加括号，但import是一个句子，不需要括号。reload是不可传递的，重载一个模块的话只会重载该模块，而不会重载该模块导入的任意模块
